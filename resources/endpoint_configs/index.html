Endpoint Configuration

This package has 4 endpoints configured for the following EPSG projections: 4326 (geographic), 3857 (webmercator), 3413 (arctic), and 3857 (antarctic).

Each endpoint is configured with NASA's Blue Marble imagery. The EPSG 4326 endpoint also has MODIS aerosol data.

For ease of use, each WMTS endpoint has an OpenLayers client for browsing the imagery.

This demo also includes a Mapserver configuration that can be used to access the demo imagery. OpenLayers endpoints are included that use Mapserver as a WMS service.
